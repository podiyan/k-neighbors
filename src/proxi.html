Proxi - find King county's closest

            /proxi/health-check : A simple health check endpoint,
                        returns : Http/200 and "ok"

            /proxi/status       : Current status of the proxi service
                        returns : index status

            /proxi/indexKML     : indexes a new kML file into proxiService
                        accepts : "path" - http path to the kml file
                        returns : http/200

            /proxi/find         : query endpoint for proxi service
                        accepts : "lat"  - lattitude of target point
                                  "long" - longitude of target point
                                  "size" - size of the resultset
                        returns : json array of placemarks
